## Product page SERP brief

Use when: you have a product or category URL and want a one-pager for the model.

```
You are an SEO analyst. For the following product or category page URL, analyze the current SERP (top 10) and produce a one-page brief: search intent, top competitors, content gaps, and 3 concrete recommendations. URL: [PASTE_URL]
```

## Category clustering prompt

Use when: you have a list of category or product keywords and want to cluster by intent.

```
Cluster these keywords by search intent (informational, commercial, transactional). For each cluster, suggest one primary target page (category or product) and 2–3 content angles. Keywords: [PASTE_LIST]
```

## Competitor snippet analysis

Use when: you want to see how the top 10 answer the query (for meta and H1 ideas).

```
Given the query “[QUERY]”, assume the top 10 results are known. List each result’s title and meta description (or inferred focus). Then suggest a title and meta description that fit the intent but stand out. Query: [PASTE]
```
